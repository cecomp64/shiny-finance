    class DebugLogger

    Defines some common logging functions for libraries. Contains a class variable called @modes, which represents the available verbosities:

    DebugLogger::modes[:silent]
DebugLogger::modes[:debug]
DebugLogger::modes[:production]

    dbg_log(str) click to toggle source

    If the mode is set to :debug, then it will print out the given string with a debug prefix

    # File debug_logger.rb, line 29
def dbg_log(str)
  if @mode != DebugLogger.modes[:debug]
    return
  end

  if @id.eql?("")
    puts("[D] #{str}")
  else
    puts("[D] [#{@id}] #{str}")
  end
end
